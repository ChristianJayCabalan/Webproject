## üîß Customization Tips

### Want to Change Colors?
1. Open `/public/css/about.css`
2. Search for `#ff9900` (orange)
3. Replace with your color
4. Repeat for other colors

### Want to Adjust Speed?
1. Find animation duration (e.g., `0.4s`)
2. Change to desired time
3. Examples: `0.3s` faster, `0.6s` slower

### Want to Disable an Animation?
1. Find the animation in CSS
2. Comment out the line starting with `animation:`
3. Done!
